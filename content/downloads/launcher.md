### Prerequisites
- [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe) file

### Guide
1. Download [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe).
2. Run it from any folder. Preferably from Desktop for easy access.
3. Wait untll all necessary files download. 
4. Select the game you want to play and click SETUP to set an existing install or download the game.
