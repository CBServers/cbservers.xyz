### Prerequisites
- Modern Warfare Remastered from Steam or [Torrent](https://github.com/CBServers/client-dowloads/raw/main/torrents/h1_full_files.torrent)
- [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe) file

### Guide
1. Download [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe).
2. Run it from any folder.
3. Click on HorizonMW tab and select SETUP.
4. Select your MWR Steam folder or torrented MWR folder.
5. Run the client by clicking on Run and allow game to update when prompted.
6. Done! Game should launch after update.
